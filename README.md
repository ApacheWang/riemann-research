# Riemann Hypothesis Research Project

## 目标

## 素数定理方向
素数定理 → 切比雪夫函数 → 黎曼ζ函数 → 零点分布 → 黎曼猜想

## 项目结构
- `src/` - Python 计算代码
- `notebooks/` - Jupyter 研究笔记
- `notes/` - 数学推导笔记
- `data/` - 计算结果数据
